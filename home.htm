Timestamp Microservice

http://pompan129-apiproj.herokuapp.com/timestamp/{DATE}
  • User Story: I can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date (example: January 1, 2016).
  • User Story: If it does, it returns both the Unix timestamp and the natural language form of that date.
  • User Story: If it does not contain a date or Unix timestamp, it returns null for those properties.

Example usage:

http://pompan129-apiproj.herokuapp.com/timestamp/July%2020,%201969
http://pompan129-apiproj.herokuapp.com/timestamp/-14256000000

Example Output:

{"unix":-14256000000,"natural":"JULY 20, 1969"}
